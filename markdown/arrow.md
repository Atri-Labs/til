# How to add arrows in Markdown?

| Activity                  |     Code     |
| :---:                     |     :---:    |
| Up arrow (↑)              |     &uarr;   |
| Down arrow (↓)            |     &darr;   |
| Left arrow (←)            |     &larr;   |
| Right arrow (→)           |     &rarr;   |
| Double headed arrow (↔)   |     &harr;   |
